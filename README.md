# Superstore Sales Dashboard (Excel)

## Project Overview

This project is an interactive Excel dashboard built using the Sample Superstore dataset. The objective of the project is to analyze sales performance, profit trends, customer segments, shipping modes, and product categories to generate meaningful business insights.

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Conditional Formatting

## Dashboard Features

* Total Sales KPI
* Average Sales KPI
* Average Profit KPI
* Category-wise Profit Analysis
* Category-wise Discount Analysis
* Region-wise Profit Distribution
* Segment-wise Sales Analysis
* Ship Mode Order Distribution
* Interactive Slicers for:

  * Category
  * Segment
  * Ship Mode

## Key Insights

* Technology generated the highest profit among all categories.
* Office Supplies received the highest share of discounts.
* The West region contributed the highest profit.
* Consumer segment generated the highest sales.
* Standard Class was the most frequently used shipping mode.

## Skills Demonstrated

* Data Cleaning and Formatting
* Data Analysis using Pivot Tables
* Dashboard Design
* Data Visualization
* Business Insight Generation
* Interactive Reporting using Slicers

## Dataset

Sample Superstore Dataset

## Project Outcome

The dashboard provides a clear overview of sales and profit performance while allowing users to interactively filter data using slicers. It helps stakeholders identify profitable categories, customer segments, regional performance, and shipping preferences for better decision-making.
